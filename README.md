# 🛍️ FormOurPlace Clone Website

A simple and responsive **FormOurPlace-inspired website** built using **HTML & CSS**. This project replicates a modern e-commerce UI with multiple pages such as cookware, bakeware, login, and a homepage.

---

## 🚀 Features

* 🏠 Clean and minimal homepage
* 🍳 Cookware page
* 🧁 Bakeware page
* 🔐 Login page UI
* 🎨 Styled with multiple CSS files
* 📱 Basic responsive design

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**

---

## 📂 Folder Structure

```
formourplace/
│
├── images/              # All images used in project
│
├── index.html           # Home page
├── cookware.html        # Cookware page
├── bakeware.html        # Bakeware page
├── login.html           # Login page
│
├── style.css            # Main styles
├── cook.css             # Cookware styles
├── bake.css             # Bakeware styles
│
└── README.md
```

---

## 📄 Pages Overview

* **index.html** → Landing / Home page
* **cookware.html** → Cookware products page
* **bakeware.html** → Bakeware products page
* **login.html** → User login interface

---

## ⚙️ How to Run

1. Clone the repository:

```
git clone https://github.com/anshp2931-gif/formourplace.git
```

2. Open the project folder

3. Run the project:

* Open `index.html` in your browser

---

## 🔗 Navigation

Make sure all pages are connected using proper links:

```html
<a href="index.html">Home</a>
<a href="cookware.html">Cookware</a>
<a href="bakeware.html">Bakeware</a>
<a href="login.html">Login</a>
```
